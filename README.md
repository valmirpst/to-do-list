# To-Do List

Um aplicativo simples e eficiente de lista de tarefas, desenvolvido com **React**, **JavaScript** e **CSS Modules**.

<br/>

## 🚀 Funcionalidades

- 📌 Criar novas tarefas
- 🗑️ Excluir tarefas
- 🔄 Reordenar tarefas
- ✅ Marcar tarefas como concluídas

<br/>

## 🛠️ Tecnologias Utilizadas

- [React](https://react.dev/)
- JavaScript
- CSS Modules
